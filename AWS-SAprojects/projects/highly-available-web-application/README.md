# Highly Available Web Application

## Overview
Designed and deployed a fault-tolerant web application across multiple Availability Zones with an Application Load Balancer and health checks to provide high availibility.

## AWS Services Used
- EC2
- Application Load Balancer
- Auto Scaling Group
- CloudWatch
- VPC

## Key Concepts Demonstrated
- High availability
- Auto-healing infrastructure
- Load balancing
- Multi-AZ deployment

## Outcome
Improved application resilience and fault tolerance through automated scaling and health monitoring.
